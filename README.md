Weather App using React + Vite, and Tailwind CSS. 

-Fetches live weather info from the free Open-Meteo API, per any searched-city.   
-Includes "Today", "Daily"(7 days) forcast, and "Hourly" for each hour of each day in the upcoming week. 
-Choose between Celsius / Km/h and Fahrenheit / Mph. 
-Fully responsive for desktop and mobile.
